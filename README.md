# 압축파일 비밀번호 크랙 소프트웨어
### 개발일자 : 2017.10

## Author: Tackhyun Jung

## Status: 완료

### 핵심목표
1) 압축파일의 비밀번호를 효과적으로 경우의 수를 조합해서 대입공격을 수행하는 코드 구현 (완료)
2) 인터페이스를 통해 입력받은 정보를 기반으로 경우의 수를 만드는 코드 구현 (완료)

---

### 사용된 기술
* 무차별 대입공격(brute-force attack)
* 경우의 수

---

### Requirement
* Python 3x
* zipfile
* itertools

---

```
> python Cracker.py
```
